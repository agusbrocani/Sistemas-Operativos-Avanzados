# Simular proyecto en Wokwi

Simula el proyecto ESP32 usando Wokwi Simulator en VS Code.

## Objetivo

Verificar que el firmware generado por PlatformIO se pueda ejecutar correctamente en Wokwi.

## Pasos

1. Verificar que exista `platformio.ini`.
2. Verificar que exista `wokwi.toml`.
3. Verificar que exista `diagram.json`.
4. Ejecutar compilacion con PlatformIO:

```powershell
pio run
